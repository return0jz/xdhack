Purpose


“In a crisis, it is acceptable to have more questions than answers. In a crisis there is no room for ‘not-invented yet’. In a crisis we should all be learners”. Through this forum we have constructed we anticipate to distinctly inform the communal about the current affairs about Covid-19. This web-portal connects you to assorted web pages bringing you up-to-date information about the pandemic. With the help of this website we’d like to share basic healthcare details allowing people to be aware the current situation.


The Portal incorporates links to websites which share key information regarding Covid-19 ranging from the number of cases per day to number of vaccinated people. It also pinpoints accessible Hospitals/Clinics; thus, assisting people. All in all the website shares facts, news and external figures about the pandemic which we believe accurately informs the audience about the highly impactful virus.
